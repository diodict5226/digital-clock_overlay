**Digital Clock Overlay**


A simple, transparent clock overlay that stays on top of all other windows. 


Features:

Always on Top: The clock floats over all other applications.

Transparent Background: Only the time (in white text) is visible.

Top-Center Position: Automatically positions itself at the top-center of the screen.

Simple Exit: Easily closed using a global hotkey.

Requirement:

Python
pip install keyboard
pip install Tkinter
